HTML Table Basics

Define: HTML Table

A table in HTML is defined using the <table> element. It consists of rows and columns with various attributes to structure and format tabular data.

When can you use these HTML elements?

  • <tr> - Can be used to define a table row.
  • <th> - Can be used to define a table header cell.
  • <td> - Should be used to define a standard table cell.
  • <thead> - Can be used to group header content in a table.
  • <tfoot> - Should be used to group footer content in a table.
  • <caption> - Should be used to provide a title or caption for a table.

What is the difference between colspan and rowspan?

Colspan is an attribute used in <th> and <td> elements to specify the number of columns a cell should span horizontally. Rowspan is an attribute to specify the number of rows a cell should span vertically.

What is the difference between cell spacing and padding?

Cell spacing refers to the space between table cells, while padding refers to the space within each cell between the content and the cell border.

How do you define table borders using CSS? How do you control border widths using CSS?

You can define table borders using CSS by applying styles to the <table>, <th>, and <td> elements. You can control border widths using the CSS border property, specifying the width, style, and color.
