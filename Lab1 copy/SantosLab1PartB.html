HTML Basics: Terms and Definitions

What is a DOCTYPE Statement?
A DOCTYPE statement is the declaration in the html that defines the specific type html being used. In other words, it basically tells the browser what type of html it needs to run and display.
How do you use: <ol>, <ul>, <dl>?

<ol> means ordered list, an example of an ordered list is a packing list.

  1. Passport
  2. Socks
  3. Tops
  4. Pants
  5. Shoes

<ul> means unordered list, an example of an unordered list is a list of 4 sided shapes

  • square
  • parallelogram
  • rectangle
  • rhombus

<dl> means definition list, an example of a definition list is:

Curiosity
The strong desire to know or learn something.
Humility
A modest or low view of one's own importance.
Sympathetic
Feeling, showing, or expressing sympathy.
What is the difference between word processor and text editor?
The main difference between a word processor and a text editor is that word processors are used for creating and editing documents. They have a variety of formatting options. An example of a word processor is Google Docs. Whereas, text editors work with plain text, like code or markup languages, such as html, css, etc.
The Client-Server Model
The Client-Server Model is a fundamental concept in computing. The client is the computer that wants services and/or resources from another computing, which is the server. The server is the compuater that provides the services and/or resources as a response to the client computer.
FTP
FTP, or File Transfer Protocol, is a network protocol that is used to transfer files between the server and the client computers. It allows these users to upload and download files.
HTTP
HTTP, also known as Hypertext Transfer Protocol, is a protocol used on the internet for transferring hypertext files. It defines how browsers and servers communicate to request and display webpages.
TCP/IP
TCP/IP, or Transmission Control Protocol/Internet Protocol, is a suite of networking protocols that allows computers to communicate over the internet. TCP handles all the reliable data transmissions. The IP handles addressing and routing.
